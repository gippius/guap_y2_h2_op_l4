# Лабораторная работа №4

Тема: "Наследование классов, базовый класс, производный класс".

В проекте реализован вариант 4: базовый класс работника больницы и производные классы `Nurse` и `Surgeon` с расчетом заработной платы.

## Сборка

```powershell
cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Запуск

```powershell
.\build\Release\lab4_hospital_workers.exe
```
